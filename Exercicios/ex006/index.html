Listas

listas ordenadas:

para fazer uma lista ordenada você usa o "OL" e depois o "li"

EX:

  1. coxinha
  2. batata
  3. harry potter
  4. mushoku tensei
  5. Tokyo ghoul
  6. Elaina supremacy

Listas não ordenadas:

Para fazer uma lista não ordenada você só precisa usar o o "UL" e dentro dele você usa o "li"

  • macarrão
  • linguiça
  • salsicha
  • batata
  • beterraba

Listas dentro de listas

Para coloca uma lista dentro de uma lista basta colocar um "OL" dentro de um "li" e fazer o processo normal.
