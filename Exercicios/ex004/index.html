Textos em negrito / ítalico

Negrito:

para usar uma frase em negrito nós temos duas formas, B e strong

A gente pode até usar o B, mas ele não é semantico

A tag "strong também deixa em negrito mas ao contrario do "b", ele é semântico.

Ítalico

Para escrever frase em Ítalica basta usar o conjunto de tags < i > mas ele não vai ser semântico

já o "EM" ele fica da mesma forma que o outro e é semantico

e para deixar deixar alguma frase marcada basta usar o comando mark

E para o mark ficar igual em todas as linhas é só colocar um style em head usando mark q funciona.

Textos sobrescritos

para escrever formulas matématicas ou escrever o nome de um elemento quimico basta usar a tag "SUB" para o número ficar em baixo e "SUP" para ficar em cima.

EX:

SUB

H2O

SUP

x5+32

Textos inseridos

Para deixar um texto sublinhado basta usar a tag "INS"
